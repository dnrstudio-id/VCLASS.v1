# LMS VCLASS (Virtual Class) 🚀

LMS VCLASS adalah platform manajemen pembelajaran berbasis web yang dirancang untuk stabilitas dan kemudahan penggunaan. Dibangun menggunakan framework **CodeIgniter 3**, sistem ini mengintegrasikan manajemen akademik, absensi mahasiswa secara real-time, dan pemantauan aktivitas sistem yang komprehensif.

## ✨ Fitur Utama

### 🛠️ Administrasi & Keamanan
- **Global Activity Logs**: Pelacakan aktivitas pengguna secara real-time dengan integrasi lintas database (Siakad-LMS).
- **System Backup**: Fitur pencadangan database per periode akademik untuk keamanan data jangka panjang.
- **Role-Based Access Control (RBAC)**: Pemisahan hak akses yang ketat antara Admin, Dosen, dan Mahasiswa.
- **Security First**: Proteksi terhadap SQL Injection, manajemen session yang aman, dan pencegahan celah keamanan web umum.

### 👨‍🏫 Fitur Dosen & Mahasiswa
- **Dynamic Attendance**: Fitur absensi mahasiswa dengan antarmuka AJAX yang responsif.
- **Social Attendance View**: Mahasiswa dapat melihat daftar hadir teman sekelas secara real-time untuk transparansi.
- **Exam Management**: Pengelolaan ujian mahasiswa dengan filter status yang dinamis.
- **Responsive UI**: Tampilan modern menggunakan Bootstrap yang nyaman diakses dari perangkat mobile maupun desktop.

## 🚀 Teknologi yang Digunakan

- **Backend**: PHP 7.4+ / 8.x
- **Framework**: CodeIgniter 3.1.13
- **Database**: MySQL (MariaDB)
- **Frontend**: Bootstrap 5, jQuery, DataTables
- **Asynchronous**: AJAX for real-time data fetching
